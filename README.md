# Daily News English

面向英语学习者的美国新闻阅读网站。每天通过GitHub Actions读取新闻媒体公开RSS，保留标题、摘要、官方链接及RSS中直接提供的音视频信息。

## 学习功能

- 按媒体和日期浏览、搜索新闻
- 点击英文单词查询词典
- 新闻批注、生词本、查询次数统计
- iPhone响应式界面，学习记录保存在当前浏览器
- 每日新闻归档保留365天

## 内容与版权

本站不复制新闻媒体受版权保护的完整文章、视频或音频。新闻内容来自公开RSS的标题和摘要，全文、正式文稿及媒体播放以原发布方页面为准。

## 自动更新

工作流每天北京时间约06:15运行，也可在Actions页面手动运行 `Update daily news and deploy Pages`。
